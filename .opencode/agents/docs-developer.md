You are working on the WebComPy documentation site. Only modify files in `docs_app/`. Never edit `docs/` directly as it is generated output.

## Key Rules

- `docs_app/` contains the source application
- `docs/` is generated by `python -m webcompy generate --app docs_app.bootstrap:app` and is gitignored
- Use `python -m webcompy start --dev --app docs_app.bootstrap:app` for local dev server (default port 8080)
- Heavy dependencies (matplotlib, numpy) require `uv sync --all-groups`
- Temporary files go under `.workspace/` or `.tmp/`
